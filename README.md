# 生命小偵探

「生命小偵探」是一款供小一視障學生使用的繁體中文生物網頁遊戲。學生在 12 個香港公園任務中，學習分辨生物與非生物、認識常見動植物、生存條件、簡單構造，以及愛護生命的方法。

## 公開遊戲

GitHub Pages 網址會在發佈後填寫於此。

## 開啟方法

作品是完全自包含的 `index.html`，沒有網絡連線時亦可使用。

1. 下載本專案。
2. 直接用 Safari、Chrome、Edge 或 Firefox 開啟 `index.html`。
3. 如需本機網站模式，可執行：

   ```powershell
   npm install
   npm run serve
   ```

4. 前往 `http://127.0.0.1:4173`。

## 遊戲規則

- 遊戲共有 12 關，每關有三個選項。
- 每題首次答對得 1 分，最高 12 分。
- 首次答錯不加分；學生會得到提示，並可重試至答對。
- 遊戲不設計時，不會自動跳到下一關。
- 重新整理頁面會重新開始，遊戲不會收集或上傳任何個人資料。

## 無障礙功能

- 頁面使用香港繁體中文語言標記及語意化 HTML。
- 支援觸控、Tab、Shift+Tab、方向鍵、Enter 和 Space。
- 所有互動元件均使用原生按鈕、單選欄位和表單群組。
- 關卡、答案提示、解釋及分數均可由螢幕閱讀器讀出。
- 題目轉換、答錯和答對後均有明確的焦點管理。
- 提供標準、大、特大三種字型。
- 提供白底黑字、黑底白字、黑底黃字三種對比模式。
- 顯示設定會保存在使用者的瀏覽器內。
- 插圖只作輔助；每題均可在看不到圖片的情況下完成。
- 沒有拖放、限時、背景音樂或自動語音。

## 自動化測試

安裝依賴及 Chromium 測試瀏覽器：

```powershell
npm install
npx playwright install chromium
```

執行全部測試：

```powershell
npm test
```

測試涵蓋：

- 首頁內容及語意結構
- 首答計分、答錯重試及 12 關完整流程
- 字型和對比設定保存
- 鍵盤操作及焦點移動
- 320px 窄螢幕、特大字型和水平捲動
- axe-core 嚴重及重大無障礙問題
- 三種顏色對比模式

## VoiceOver 實機驗收清單

裝置：iPad、Safari、VoiceOver。

- [ ] 由頁首逐項掃讀至「開始遊戲」，順序與畫面一致。
- [ ] 字型及對比單選欄位會讀出名稱和選取狀態。
- [ ] 每關會讀出關卡、學習主題、插圖描述、題目和選項。
- [ ] 選擇答案後可找到並啟動「提交答案」。
- [ ] 答錯會將焦點移到提示；其後可找到「再試一次」。
- [ ] 答對會讀出解釋，焦點落在「下一關」。
- [ ] 完成後會讀出標題、分數、鼓勵語和學習重點。
- [ ] 不會出現無法離開的焦點區域或只靠顏色表達的資訊。

## NVDA 實機驗收清單

裝置：Windows、Chrome 或 Firefox、NVDA。

- [ ] 使用 H 鍵可依合理次序瀏覽所有標題。
- [ ] 使用 F 鍵可找到遊戲表單，R 鍵可逐項瀏覽答案。
- [ ] 使用 Tab、Shift+Tab、方向鍵、Space 和 Enter 可完成遊戲。
- [ ] 即時分數、答錯提示和答對解釋只會讀出一次。
- [ ] 切換關卡後，NVDA 會讀出新的關卡標題。
- [ ] 三種字型和三種對比模式均可正常操作。

實機測試後，請在方格加上剔號並在功課記錄所用裝置、瀏覽器和日期。

## GitHub Pages 發佈

1. 將專案推送到 GitHub 公開儲存庫。
2. 在儲存庫開啟 `Settings` → `Pages`。
3. `Source` 選擇 `Deploy from a branch`。
4. 選擇 `main` 分支及根目錄 `/ (root)`，然後儲存。
5. 等候 GitHub 顯示公開網址，再以無痕視窗確認毋須登入。

如要放入 Google Sites，可選擇「插入」→「嵌入」→「網址」，貼上 GitHub Pages 網址。

## 檔案

- `tests/`：Playwright 遊戲流程和無障礙測試。
- `playwright.config.js`：測試伺服器及瀏覽器設定。
- `package.json`：測試工具及本機伺服器指令。
